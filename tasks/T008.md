## Section Requirements

### 1. Hero Section

- Large headline: "Building the Future of Intelligent Software"
- Subheadline: Brief value proposition
- Primary CTA: "Book a Free Consultation"
- Secondary CTA: "View Our Work" (links to projects)
- Background: Dark with subtle gradient or pattern
- Animation: Text reveal with Framer Motion

### 2. Value Proposition Section

- 3-4 key differentiators with icons
- Trust indicators (years in business, projects completed, etc.)
- Brief paragraph about the company
- Stats or metrics (if available)

### 3. Services Preview

- Grid of 4-6 service cards
- Each card: icon, title, brief description
- Link to full services page
- Hover animations

### 4. Process Section

- 4-step process: Discovery → Design → Build → Launch
- Visual timeline or numbered steps
- Brief description for each step
- Builds trust by showing methodology

### 5. Projects Showcase

- 2-4 featured projects
- Project cards with images
- Tech stack tags
- Links to project detail pages
- Note: Can use static data initially, will integrate with Sanity later

### 6. Testimonials

- 2-3 client testimonials
- Quote, client name, company
- Can use placeholder data for now

### 7. CTA Section

- Strong headline: "Ready to bring your idea to life?"
- Subtext explaining the consultation
- Large CTA button
- May include brief benefits list

### 8. FAQ Section (Enhance Existing)

- 5-6 relevant questions
- Accordion-style expand/collapse
- Questions about process, pricing, timeline
